Brand Colors

This design system uses a very minimal color palette as the setting for my different case studies, which will each have a signature color. The portfolio's main staples are Solid Black, Off-White, and Seafoam Green.


Solid Black

Hex Code: 000000

This color is used for all typefaces, logos, and in the outlines of the buttons.

Off-White

Hex Code: FFFFFD

This color is used for the background for the portfolio website.

Seafoam Green

Hex Code: 60D8B4

This color aids in hovering over links and indicating which page the site visitor is viewing.
