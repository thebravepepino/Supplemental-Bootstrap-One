Buttons

These are the variations of the button that will be used on the portfolio site.

Usage

Buttons in portfolios should be only used when necessary. In this case, it wasn't particularly necessary to add a button, but it shows that I am capable of coding this type of component into projects.

Accessibility Considerations

I thought about including an additional button on the home page to "View Work" that would take you to the UX Case Study page, but I decided against this approach because it might cause confusion to the user about the difference between clicking the UX Case Study navigation link and the second button.
